# OfferFlow Frontend

React + TypeScript single-page application built with Vite.

## Prerequisites

- Node.js 18+ and npm

## Scaffold (one-time)

This folder is a placeholder. Generate the Vite app **into this folder** by running from
the `frontend/` directory:

```powershell
npm create vite@latest . -- --template react-ts
```

When prompted that the directory is not empty, choose to keep the existing `README.md`.

## Setup

```powershell
# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be served at http://localhost:5173.
